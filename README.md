# MessageLink
MessageLink is an encrypted messaging scheme built for application in a client-server architecture.

# PyCharm IDE Setup

## Assumptions:
- You have a version of the JetBrains PyCharm IDE installed (either Professional or CE)
  
## Get started developing by executing the following commands:
Note: Be sure to first navigate to wherever you'd like to store the repo locally.

#### Via GitHub CLI
Note: If using this method, be sure to download the CLI binaries via the following link: https://cli.github.com

```
gh auth login
gh repo clone mcoryell/MessageLink
```

#### Via Git CLI

```
git clone https://github.com/mcoryell/MessageLink.git
cd MessageLink
```

### Editing in PyCharm

1. Open PyCharm IDE
2. File -> Open... -> Navigate to wherever you stored the repo and open the entire folder

# You're all set to develop!
